Grid System

The grid system within our D3.js pattern library project serves as the fundamental framework for organizing page layouts. It leverages a set of class definitions, including .containers, .row, and .col, to efficiently structure and align content. Employing the power of flexbox, this grid system seamlessly adapts to different screen sizes, ensuring full responsiveness. It is designed to support a maximum of 12 columns for versatile layout options.

Setting One Column Width

Auto-layout for flexbox grid columns also means you can set the width of one column and have the sibling columns automatically resize around it. You may use predefined grid classes (as shown below), grid mixins, or inline widths. Note that the other columns will resize no matter the width of the center column.

Images

All images within the web page should be designed and coded to be responsive, adapting seamlessly to the varying screen sizes and breakpoints. This means that regardless of the user's device, whether it's a small smartphone, a medium-sized tablet, a large desktop monitor, or anything in between, the images on the page should scale, resize, and adjust their dimensions to fit the available screen real estate appropriately.

Note that to check if the image on your screen is responsive, resize the window of your computer.

Breakpoints

Creating web application dimensions for different screen sizes. Breakpoints will be used to create flexible interfaces regardless of whether the site is viewed on mobile, tablets, or desktop.

Core Concepts

Responsive designs have six default breakpoints (grid tiers). These are the dimensions of the identified screen sizes.

Default Breakpoints
Breakpoint Class Infix Dimensions
Extra Small None <576px
Small sm >576px
Medium md >768px
Large lg >992px
Extra Large xl >1200px
It is important to note that the breakoint widths are multiples of 12 and can be customized via Sass - in a Sass map in the _variables.scss stylesheet.

Breakpoints are supported by a few media queries that will adjust the age depending on the viewport size.

Viewport Breakpoint Notes
Min-width
  • @include media-breakpoint-up(sm){..}
  • @include media-breakpoint-up(md){..}
  • @include media-breakpoint-up(lg){..}
  • @include media-breakpoint-up(xl){..}
  • @include media-breakpoint-up(xxl){..}
For xs breakpoint, @media (min-width:0) {...} is used
Max-width
  • @include media-breakpoint-down(sm) {...}
  • @include media-breakpoint-down(md) {...}
  • @include media-breakpoint-down(lg) {...}
  • @include media-breakpoint-down(xl) {...}
  • @include media-breakpoint-down(xxl) {...}
For xs breakpoint, @media (max-width: 0) {...} is used.
